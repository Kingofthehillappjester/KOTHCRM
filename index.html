How to host this as a shared URL
1

Download the HTML file

Save index.html to your computer (or download it from Claude). This is your entire app — one file.

2

Create a free GitHub account & repo

Go to github.com → New repository → name it koth-crm → Public → Create. Then upload index.html.

3

Enable GitHub Pages

In the repo → Settings → Pages → Source: Deploy from branch → Branch: main → Save. Your URL will be: https://yourusername.github.io/koth-crm

4

Create a Google Sheet & API key

Create a blank Google Sheet. Share it with both of you (Editor). Get a free Sheets API key from Google Cloud Console → enable the Sheets API → create an API key. Each person enters their own API key when they open the app.

5

Send Matt the URL

Share the GitHub Pages URL with Matt. He opens it, picks his name, enters the same Sheet ID, and his own API key. You're both live on the same pipeline.
